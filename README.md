# 📚 Лабораторные работы 

Здесь собраны мои университетские лабораторные работы.

## 📁 Структура репозитория

*   **`linear_algebra/`** — Лабораторные работы по линейной алгебре.
    *   `linear_algebra_lab_1.ipynb` — изучение библиотек для работы с матрицами, анализ скорости и точности вычислений.
*   **`calculus/`** — Лабораторные работы по математическому анализу.
    *   `calculus_lab_1.ipynb` — применение Python для решения задач по математическому анализу.

## 🚀 Запуск проектов

Для работы с ноутбуками Jupyter потребуется Python и несколько библиотек.

1.  **Установите зависимости** (если необходимо)

2.  **Запустите Jupyter Notebook** из папки с проектом

3.  В открывшемся браузере выберите нужный файл `.ipynb` для просмотра и запуска кода.

## ✍️ Примечание
Все работы выполнены в среде Jupyter Notebook. GitHub поддерживает их интерактивный просмотр прямо на сайте.
