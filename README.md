# Лабораторные работы 

Здесь собраны мои университетские лабораторные работы.

## Структура репозитория

*   **`linear_algebra/`** — Лабораторные работы по линейной алгебре.
    *   `linear_algebra_lab_1.ipynb` — изучение библиотек для работы с матрицами, анализ скорости и точности вычислений. Дополнительный файл в папке `linear_algebra/data_1/`.
    *   `linear_algebra_lab_2.ipynb` — реализация линейной регрессии (МНК), демонстрация переобучения и применение L2-регуляризации для повышения устойчивости модели. Данные для анализа находятся в папке `linear_algebra/data_2/`.

*   **`calculus/`** — Лабораторные работы по математическому анализу.
    *   `calculus_lab_1.ipynb` — применение Python для решения задач по математическому анализу.
    *   `calculus_lab_2.ipynb` — интерполяция данных с анализом устойчивости методов, моделирование пороговой криптографии (Шамир) и исследование многомерных функций.
